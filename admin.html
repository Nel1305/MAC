Comment fonctionne le stockage

Toutes les données (commandes, réservations, messages, newsletter, albums, événements) sont stockées dans Netlify Blobs, directement sur votre hébergement Netlify.

Aucune configuration externe n'est nécessaire — Netlify gère tout automatiquement.

Pour inspecter ou supprimer des données via la CLI :

netlify blobs:list commandes

netlify blobs:delete commandes <id>

Pour consulter visuellement : app.netlify.com → votre site → Blobs

Configuration EmailJS

1. Créez un compte gratuit sur emailjs.com

2. Email Services → Add Service → Gmail → Connect Account

3. Email Templates → Create → utilisez ces variables :
{{name}} {{time}} {{message}}

4. Account → API Keys → copiez la Public Key

5. Collez les 3 valeurs dans script.js → section EMAILJS_CONFIG

Variable d'environnement sécurité

Pour protéger l'API admin, définissez ADMIN_SECRET dans Netlify :

app.netlify.com → votre site → Site configuration → Environment variables → Add variable

Nom : ADMIN_SECRET — Valeur : votre mot de passe admin (mac2025! ou autre)

Ce secret est vérifié par les Functions pour autoriser les lectures/écritures sensibles.
